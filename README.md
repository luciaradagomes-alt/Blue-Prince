# Blue-Prince

Jeu simple basé sur le jeu Blue Prince.

## Prérequis

- Python 3
- pip

## Installation

1. Cloner le dépôt
    ```sh
    git clone https://github.com/luciaradagomes-alt/Blue-Prince/
    ```

2. Créer un environnement virtuel
    ```sh
    virtualenv blueprince
    ```

3. Installer les dépendances
    ```sh
    pip install -r requirements.txt
    ```

## Utilisation

    ```sh
    python game.py
    ```
